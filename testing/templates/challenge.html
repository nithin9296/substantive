Frequently asked questions

Which type of Audit can I use this for

Shrota application can be used for Financial Audits, Operational Audits, Internal Audits and Tax audits.

What are the steps

Singup, start membership, create client profile, upload samples and share the details to client so that he can upload supportings against samples selected.

How many client profiles can I create?

Currently there is no limit on number of client profiles that one can create.

What is this application meant for?

It is an Workspace for testing samples during audit, this is the first module out of the series of the modules to be launched.

Why do I need to input my credit card for the free trial?

We require credit card information for the trial to prevent fraud. But if you cancel before the trial period is over, there won’t be any charges.
